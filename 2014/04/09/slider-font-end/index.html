Custom Icon Font

  • icomoon
  • fontello
  • iconfont

优点

  • 分辨率无关
  • 文件小
  • 加载性能
  • 支持CSS样式：可以利用CSS来定义大小、颜色、阴影、hover状态、透明度、渐变...
  • 兼容性好：老的移动端浏览器，Android 2.1以下 Opera mini

缺点

  • 颜色样式单一
  • unicode 冲突: PUA安全区域的字符编码 U+E000..U+F8FF

注意

  • 图形需要闭合，不要用stroke，否则会显示不出来
  • 图形合并并扩展
  • 用单色
